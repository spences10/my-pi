# @spences10/pi-themes

Theme pack for the Pi coding agent.

## Install

```bash
pi install npm:@spences10/pi-themes
```

Then choose a theme in `/settings`, or persist one in Pi settings
JSON:

```json
{
	"theme": "tokyo-night"
}
```

## Included themes

- Catppuccin Mocha
- Dracula
- Gruvbox Dark
- Night Owl
- Nord
- One Dark
- Rosé Pine
- Solarized Dark
- Tokyo Night
